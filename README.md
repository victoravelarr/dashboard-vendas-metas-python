# dashboard-vendas-metas-python
Dashboard de Vendas e Atingimento de Metas

Descrição:

Este projeto consiste no desenvolvimento de um dashboard interativo de vendas utilizando Python, com foco na análise de faturamento, desempenho por produto e acompanhamento do atingimento de metas mensais.

O dashboard permite a aplicação de filtros dinâmicos por ano, mês e produto, atualizando automaticamente todas as métricas e visualizações apresentadas.

Objetivos:

- Analisar o faturamento total e mensal

- Comparar o faturamento realizado com a meta mensal

- Identificar produtos que mais contribuem para o faturamento

- Avaliar o percentual de atingimento das metas

- Criar visualizações claras para apoio à tomada de decisão

Tecnologias Utilizadas:

- Python

- Pandas

 -Streamlit

- Plotly Express

- Excel (Base de dados)

Funcionalidades:

Filtros interativos por:

- Ano

- Mês

- Produto

Métricas principais:

- Faturamento total

- Quantidade vendida

- Ticket médio

- Percentual médio de atingimento da meta

Visualizações:

- Comparativo de faturamento versus meta mensal

- Evolução das vendas ao longo do tempo

- Distribuição do faturamento por produto

Estrutura dos Dados:

A base de dados contém informações simuladas de vendas, incluindo data, produto, quantidade vendida, valor unitário e meta mensal.
O faturamento é calculado dinamicamente a partir da quantidade vendida e do valor unitário.

Como Executar o Projeto

1- Instale as dependências:

pip install pandas streamlit plotly openpyxl


2- Execute o dashboard:

streamlit run main.py

Observação

Projeto desenvolvido com foco em aprendizado prático em análise de dados e visualização, simulando um cenário real de negócio.
